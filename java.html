Lý thuyết Java

  • Java là một ngôn ngữ lập trình hướng đối tượng phổ biến, được phát triển bởi Sun Microsystems vào năm 1995 (hiện thuộc sở hữu của Oracle). Java nổi bật với khẩu hiệu "Write once, run anywhere" (Viết một lần, chạy mọi nơi), nhờ vào cơ chế Java Virtual Machine (JVM) cho phép chương trình Java chạy được trên nhiều nền tảng khác nhau mà không cần sửa đổi mã nguồn./li>
  • Tính năng:
  • Đa nền tảng (Cross-platform): Nhờ JVM, Java có thể chạy trên Windows, macOS, Linux...
  • Lập trình hướng đối tượng (OOP): Hỗ trợ đầy đủ class, object, kế thừa, trừu tượng, đa hình.
  • Thu gom rác tự động (Garbage Collection): Quản lý bộ nhớ tự động.
  • Object-Oriented Programming
  • Kiến thức nền tảng cần nắm trong Java:
  • Cấu trúc chương trình Java cơ bản (main(), import, package)
  • Câu lệnh điều kiện và vòng lặp
  • Lập trình hướng đối tượng
  • Sử dụng thư viện chuẩn (Java API)
